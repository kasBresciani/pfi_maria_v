===== Preparação do ambiente =====

> Antes de instalar as dependências, é necessário ter o node.js intalado na sua máquina.
> Seu download é encontrado em: https://nodejs.org/en

| Instalando dependências:

1 - Com a pasta do projeto aberto, abra um terminal de comando integrado (botão direito em baixo do ultimo arquivo > Abrir no terminal integrado)

2 - Quando a janela de terminal integrado abrir, execute o comando "npm install"

- Agora todas as dependências estarão instaladas

| Configurando o banco de dados no projeto:

1 - No MySQL Workbench, crie o banco de dados normalmente (CREATE DATABASE _nomeDoBanco_)

2 - Abra o arquivo app.js

3 - No arquivo, conigure a URL de conexão

const sessionStore = new MySQLStore({
  host: 'localhost',
  user: 'NOME DO USUARIO',
  password: 'SENHA DO BANCO',
  database: 'NOME DO BANCO DE DADOS'
});

4 - Abra o arquivo config > database.js

5 - Configure a URL no arquivo aberto:

const sequelize = new Sequelize('NOME DO BANCO', 'NOME DO USUARIO', 'SENHA DO BANCO', {
    host: 'localhost',nmp
    dialect: 'mysql'
});


-> Essas informações são definidas na hora da instalação do banco, por isso é importante sabê-las.
-> A URL normalmente é 127.0.0.1, mas pode usar localhost

| Criando o banco de dados:

1 - Fiz um esquema de migrations. Dessa maneira, ao executar o projeto pela primeira vez,
 todas as tabelas serão criadas automaticamente.

===== Executando o projeto =====

1 - Abra o arquivo "app.js".

2 - Execute-o normalmente.

3 - A URL do site aparece-rá no terminal da aplicação.

<localhost:3000>

Qualquer dúvida, fico a disposição!
